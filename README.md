# Securing APIs using OAuth 2.0 with GitHub and Auth0

## Description
This project demonstrates OAuth 2.0 authentication using GitHub and Flask. It includes login, routes, and session-based authorization.

## Features
- GitHub OAuth Login
- Protected API
- Session authentication
- Logout functionality

## Tech Stack
- Python
- Flask
- Authlib
- GitHub OAuth

## Pages
- /login
- /callback
- /profile (Authorize)
- /logout
- /api/secure-data (Authorize)

## How to Run
1. Install dependencies:
   pip install flask authlib requests

2. Run the app:
   python app.py

3. Open:
   http://localhost:5000/login
